> [!WARNING]
> This project is in early development and is not yet ready for production use.

## Outline

## Build

### Clone the repository

```sh
git clone https://github.com/appcypher/prompt-bar.git
```

### Build the UI

```sh
cd client
```

```sh
pnpm install
```

### Start the UI

```sh
cargo tauri dev
```
